1. **src/hooks/useChat.ts**: Create the orchestration hook that wires store, API, and context:
   ```typescript
   export function useChat(agentId: AgentId = 'diana') {
     // Return: { messages, isStreaming, error, tokenCount, sendMessage, cancelStream, retryLastMessage, clearError }
   }
   ```
   Implementation:
   - On mount: call `useChatStore.getState().loadConversations()`, then `getOrCreateConversation(agentId)` to set activeConversationId
   - `sendMessage(content: string)`:
     1. Clear any existing error
     2. Add user message to store via `addMessage`
     3. Get current conversation from store
     4. Create AbortController, call `startStreaming(controller)`
     5. Call `sendStreamingMessage(agentId, messages, callbacks, controller.signal)` where:
        - `onToken`: accumulate content, call `updateStreamingContent`
        - `onComplete`: call `stopStreaming`, add assistant message via `addMessage`, update token count via `updateConversationTokens`. Then check if summarization needed: if conversation.totalTokens > SUMMARIZE_THRESHOLD * TOKEN_LIMITS[DEFAULT_MODEL], call `summarizeConversation`
        - `onError`: call `stopStreaming`, call `setError` with user-friendly message
   - `cancelStream()`: call `streaming.abortController?.abort()`, then `stopStreaming()`
   - `retryLastMessage()`: remove last assistant message (if it was an error), re-send with previous user message
   - Derive `messages` from active conversation in store
   - Derive `isStreaming` from store streaming state
   - Derive `tokenCount` from active conversation's totalTokens
   - Derive `error` from store error state
