- Visible head (4x4 px), body (6x8 px), legs (distinct in walk frames)

- Walk frames: leg alternation pattern (frame 0=stand, 1=left step, 2=stand, 3=right step)
- Work frames: arm movement at desk (typing motion)

**renderer.ts changes:**

Replace `renderCharacter()`:
- Import CHARACTER_ATLAS from spriteAtlas, getCharacterSheet from spriteSheet

- If characterSheet is loaded, use `getCachedSprite(characterSheet, frame, zoom)` and `ctx.drawImage(cached, x, y)` instead of fillRect
- If characterSheet is null (not loaded yet), fall back to existing colored rectangle rendering
- Remove the direction indicator triangle -- sprites show direction natively

Replace `getTileColor()` and tile rendering in `renderFrame()`:
- Import ENVIRONMENT_ATLAS from spriteAtlas, getEnvironmentSheet from spriteSheet

  - Use `getCachedSprite(environmentSheet, frame, zoom)` and `ctx.drawImage()`
  - Fall back to colored rectangles if environmentSheet is null
- Remove the grid line strokeRect for tiles (sprites have their own visual edges)
