<task type="auto">
  <name>Task 1: Create glow effects module + day/night system</name>
  <files>src/engine/glowEffects.ts, src/engine/timeOfDay.ts</files>
  <action>
Create `src/engine/glowEffects.ts`:

1. Define `GlowSource` interface: `{ x: number; y: number; radius: number; color: string; type: 'monitor' | 'lamp' | 'ambient'; pulse: boolean; }`.

2. Build `GLOW_SOURCES` array by scanning FURNITURE + DECORATIONS from officeLayout.ts:
   - For each room's desk, place a monitor glow source at desk center (col * TILE_SIZE + width * TILE_SIZE / 2, row * TILE_SIZE + TILE_SIZE / 2). Color: `'100, 149, 237'` (cornflower blue). Radius: `TILE_SIZE * 2`. Type: `'monitor'`, pulse: true.
   - For each room's desk (EXCEPT war-room), place a lamp glow source offset 1 tile to the side of the desk. Color: `'255, 191, 64'` (warm amber). Radius: `TILE_SIZE * 2`. Type: `'lamp'`, pulse: false.
   - War Room: single monitor glow centered on the conference table area (col 15.5 * TILE_SIZE, row 17 * TILE_SIZE). No desk lamps per user decision.
   - Hallway: no explicit hallway sources -- let office glow radii spill through doors naturally.
   - Rec area: warm ambient glow at water cooler position (col 10, row 23). Color: amber. Type: `'ambient'`, pulse: false.

3. Export `renderGlowEffects(ctx, timeOfDay, elapsedTime)`:
   - `ctx.save()`, set `globalCompositeOperation = 'lighter'`.
   - For each source in GLOW_SOURCES:
     - Compute alpha: base 0.25 for monitors, 0.2 for lamps/ambient.
     - If pulse: multiply by `0.85 + 0.15 * Math.sin(elapsedTime * Math.PI / 2)` (~4s full cycle).
     - Apply day/night: multiply alpha by `1.0 - timeOfDay * 0.8` (glows dimmer in day).
     - Create radial gradient from source center (inner radius 0) to outer radius.
     - Inner color stop: `rgba({color}, {alpha * 0.3})`. Outer: `rgba({color}, 0)`.
     - Fill rect covering glow bounds.
   - `ctx.restore()`.

Create `src/engine/timeOfDay.ts`:

1. Export `computeTimeOfDay(): number` -- returns 0.0 (night) to 1.0 (day):
   - Get fractional hour: `new Date().getHours() + new Date().getMinutes() / 60`.
   - Dawn 6-7: `(h - 6)`. Day 7-18: `1.0`. Dusk 18-19: `1.0 - (h - 18)`. Night: `0.0`.
   - Cache result for 10 seconds (avoid recalculating every frame per research anti-pattern).

2. Export `applyFloorTint(ctx, timeOfDay, bounds)`:
   - If timeOfDay > 0.5: warm amber overlay `#ffd700` at alpha `(timeOfDay - 0.5) * 0.06` (max 3%).
   - If timeOfDay <= 0.5: dark blue-black `#000020` at alpha `(0.5 - timeOfDay) * 0.15` (max 7.5%).
   - Use ctx.save/restore with globalAlpha + fillRect.
  </action>
