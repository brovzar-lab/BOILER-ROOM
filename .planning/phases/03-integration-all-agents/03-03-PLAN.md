**OverviewPanel.tsx — Agent cards grid for BILLY's office:**

Create a new component that displays when BILLY is at his own office (activeRoomId === 'billy' or null).

Structure:
- Title: "Command Center" in amber/gold text
- Subtitle: "Your agents are standing by" in muted text
- Grid of 5 agent cards

Each agent card shows:
- Left color accent bar (4px wide, agent's color from PLACEHOLDER_COLORS or persona config)
- Agent name (bold) and title (muted)
- Status indicator: green dot for 'idle', amber pulsing dot for 'thinking', red dot for 'needs-attention'
- Last message preview: truncated to ~60 chars from the most recent assistant message in that agent's conversation. Show "No conversation yet" in muted italic if no messages.
- Last activity timestamp (relative: "2m ago", "1h ago") from conversation.updatedAt. Use simple relative time helper (don't import a library).

Card interaction: `onClick` triggers BILLY walking to that agent's room:
```typescript
import { startWalk } from '@/engine/characters';
import { OFFICE_TILE_MAP, ROOMS } from '@/engine/officeLayout';

function navigateToAgent(agentId: string) {
  const room = ROOMS.find(r => r.id === agentId);
  if (!room) return;
  const state = useOfficeStore.getState();
  state.setTargetRoom(room.id);
  startWalk('billy', room.billyStandTile.col, room.billyStandTile.row, OFFICE_TILE_MAP);
}
```

Card hover: subtle border highlight using agent's color at low opacity.

Layout: Use CSS grid. Per user discretion, use a single-column layout for the 400px panel width — 5 stacked cards with small gaps. Each card is a compact horizontal strip (not a large square card) to fit all 5 without scrolling.

Data sourcing:
- Agent personas: import `agents` from `@/config/agents` and iterate `Object.values(agents)`
- Agent statuses: `useOfficeStore((s) => s.agentStatuses)`
- Conversations: `useChatStore((s) => s.conversations)` — find conversation where `conv.agentId === agent.id`

Style: Dark theme consistent with rest of app. Cards use `bg-[--color-surface-card]` or similar dark surface. Text uses `--color-text-primary` and `--color-text-muted`.

Keyboard shortcut hint: small text at bottom "Press 1-5 to navigate" in muted color.
