    **Step 1: Create `src/components/canvas/OfficeCanvas.tsx`.**
    React component that mounts and manages the canvas:
    ```
    - useRef<HTMLCanvasElement> for canvas DOM reference
    - useEffect on mount:
      1. Get canvas element from ref
      2. HiDPI setup: read devicePixelRatio, set canvas.width/height to CSS dims * DPR,
         set canvas.style.width/height to CSS dims, ctx.scale(dpr, dpr), ctx.imageSmoothingEnabled = false
      3. Set CSS `image-rendering: pixelated` on canvas element
      4. Call loadAllAssets() (awaits sprite loading before starting loop)
      5. Call useOfficeStore.getState().initializeCharacters()
      6. Call startGameLoop(canvas) -- store cleanup function
      7. Call setupInputHandlers(canvas) -- store cleanup function
      8. Return cleanup: call both cleanup functions
    - ResizeObserver on canvas parent: when container resizes, re-run HiDPI setup
      (update canvas.width/height, re-scale context)
    - Render: <canvas> element with className for full-size positioning
    ```
    The canvas should be positioned absolutely to fill its parent container.
    Use `w-full h-full absolute inset-0` Tailwind classes.
